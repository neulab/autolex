AutoLEX: Language Descriptions Explorer

All of the world's languages have an adherence to a set of principles specifying how to create sentences in the language. AutoLEX is a tool for exploring the different linguistic phenomena automatically extracted from raw text in a concise, human-and machine-readable format.

Here are the languages (and treebanks) we currently support.
